# Project Title
Credit Card

## Description
Credit Card Animation Page where user can input their credit card details like Card Number, Card Name and Expiry Details with number formatting, validation, and automatic card-type detection.

## Getting Started

### Dependencies

* Node Js >= 16

### Installing
```bash
npm install
```

## Deployment
### DEV environment
```bash
npm start
```

### To build
```bash
npm run build
```
